 🚚 Анализ рисков и визуализация логистических цепочек поставок
Проект посвящен полному циклу работы с логистическими данными: от первичной очистки и поиска скрытых зависимостей до создания интерактивного бизнес-инструмента для мониторинга рисков.
 🔍 Исследовательский анализ данных (EDA)
В среде 'Jupyter Lab' была проведена комплексная работа с сырыми логистическими данными (файл `[01_data_cleaning].ipynb`)
Очистка данных: удалены дубликаты, обработаны пропущенные значения и приведены к нужным форматам временные метки этапов доставки.
Анализ участников: детально изучены все звенья цепочки поставок (порты отправления, перевозчики,порты назначения ) и оценена эффективность их работы.
Поиск зависимостей: выявлены ключевые факторы, которые сильнее всего влияют на общее время груза в пути.
Работа с выбросами: обнаружены и проанализированы аномальные задержки, которые искажали общую статистику.
 📊 Интерактивный дашборд (Power BI)
На основе инсайтов из EDA был разработан многостраничный дашборд в **Power BI** (файл `[LOGShipments_project].pbix`). Он содержит многочисленные графики и срезы для сквозного контроля логистики.

Ключевые возможности дашборда:
1. 'Контроль этапов': визуализация прохождения груза по всей цепочке.
2. 'Мониторинг рисков': автоматическая подсветка узких мест, где чаще всего происходят сбои.
3. 'Прогнозирование задержек': графики позволяют наглядно оценить вероятность опоздания доставки на основе исторических данных.
4. 'Гибкие фильтры': данные можно фильтровать по конкретным участникам поставок, портам отправления и прибытия, типам транспорта и временным периодам.
 🛠️ Стек технологий и инструменты
 -Python (Pandas, NumPy, Matplotlib, Seaborn) — очистка, фильтрация данных и статистический анализ.
 -Jupyter Lab — среда для проведения EDA.
 -Power BI & DAX — проектирование модели данных, создание мер и интерактивных отчетов.
 -Git / GitHub — контроль версий и публикация проекта.
